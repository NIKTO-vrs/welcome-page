💣 TheFatRat 💣

⚔️ درباره ابزار: TheFatRat یک ابزار فوق‌العاده برای ساخت backdoor مخرب در سیستم‌های ویندوز و اندروید است. این ابزار می‌تواند فایل‌های اجرایی مخرب (malware) تولید کند که توسط بسیاری از آنتی‌ویروس‌ها شناسایی نمی‌شوند.

🔥 ویژگی‌ها:

  • ساخت فایل‌های مخرب برای ویندوز، اندروید، لینوکس
  • پشتیبانی از متاسپلویت
  • بایپس آنتی ویروس
  • رابط کاربری ترمینال آسان و سریع

⚙️ دستورات نصب:

git clone https://github.com/Screetsec/TheFatRat.git
cd TheFatRat
chmod +x setup.sh
./setup.sh
